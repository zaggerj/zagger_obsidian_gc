# 安装nerd font

　　1.1、安装nerd font  
因为以上字体都会缺少符号，nerd font是一个非常棒的字体库，而且对p10k的兼容性非常好，如果不想费事的话，强烈推荐安装这个。

# git 克隆字体库

　　git clone https://github.com/ryanoasis/nerd-fonts.git --depth 1

# 打开文件夹

　　cd nerd-fonts

# 安装字体

　　./install.sh
